## APEX Metric Definition

The **APEX Metric** is the SINGLE aggregated weighted outcome combining all six category subtotals:

```
APEX = Subtotal_ALPHA(30%) + Subtotal_VELOCITY(25%) + Subtotal_SHIELD(15%) + Subtotal_EFFICIENCY(15%) + Subtotal_CONTINUITY(10%) + Subtotal_MARKET_SHARE(5%)
```

- **6 Category Subtotal Metrics** (one per pillar)
